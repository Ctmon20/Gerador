📖 Como usar o Gerador?

  • Escolha a Loja: Isso mudará automaticamente a cor do card e o texto do botão (Shopee, Amazon ou Mercado Livre).
  • Imagem: Cole o link de uma imagem do produto. O card irá atualizar em tempo real.
  • Título: Escreva o nome do produto e o preço. Use emojis para chamar mais atenção!
  • Link: Insira o seu link de afiliado personalizado.
  • Download: Clique no botão azul para salvar a imagem pronta para postar no Instagram ou WhatsApp.
  • Texto: Use o botão verde para copiar a legenda já formatada com seu link.
